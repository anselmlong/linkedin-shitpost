# .impeccable.md — LinkedIn Shitpost Generator

## Design Context

### Users
Personal project. No defined user persona — it's a joke app for anyone who enjoys mocking LinkedIn culture. Users arrive wanting to generate satirical LinkedIn posts quickly and laugh at the output. The experience should be fast and funny, not onboarding-heavy.

### Brand Personality
**Three words: polished, sardonic, deadpan.**

The comedy is entirely in the contrast — the UI is sincere LinkedIn, the content is absurd. No winking at the camera in the interface itself. The straighter the face, the funnier it is.

### Aesthetic Direction
- **Light mode only.** LinkedIn is light. We are LinkedIn (aesthetically).
- **LinkedIn's actual design language:** `#F3F2EF` page background, white cards, `#0A66C2` primary blue, `#E0DFDC` borders, `#191919` text, system font stack.
- **Anti-reference:** The old dark/gradient "vibe-coded AI tool" aesthetic. No neon, no dark bg, no gradient cards, no custom brand identity.
- **Reference:** linkedin.com itself — feed, post composer, card layouts.

### Design Principles
1. **Straight face.** The UI does not joke. It looks exactly like LinkedIn. The comedy is the content's job.
2. **LinkedIn patterns, not custom ones.** When in doubt, ask: does LinkedIn do this? Use their border radius, their colors, their button shapes.
3. **Restraint over decoration.** No gradient headers, no colored card borders for personas (except the two featured AI picks), no emoji-heavy chrome.
4. **Fast to the joke.** The input is minimal (text + one button labeled "Shitpost"). Get out of the way so the user can read the posts.
5. **Fake posts, real layout.** Generated posts render as convincing LinkedIn post cards — avatar, name, engagement bar — because the illusion is the point.
